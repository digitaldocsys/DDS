# Digital Document System (DDS)

**Organize. Index. Retrieve.**

Digital Document System (DDS) is a lightweight document management solution designed for small and medium businesses that handle large volumes of paper documents.

Instead of spending valuable time searching through filing cabinets, shared folders, emails, and archive boxes, DocDex helps users organize and retrieve documents in seconds.

---

## 🆕 NEW! OCR Auto Document Classification

DDS now supports **OCR (Optical Character Recognition)** to automatically read document contents and assist with document classification.

Benefits include:

- 📄 Automatically recognize text from scanned PDFs and images
- 🤖 Assist users in classifying documents
- 🏷️ Reduce manual indexing effort
- ⚡ Improve indexing speed and accuracy
- 🔍 Make document organization even easier

*OCR is designed to assist document classification and indexing, helping users process large volumes of documents more efficiently.*

## How It Works

### 1. Organize
Import your existing PDF or image files into DDS.

### 2. Index
With OCR-assisted classification, index each document using searchable information such as customer name, invoice number, document type, or year.

### 3. Retrieve
Locate any document within seconds using fast and flexible search capabilities.
